# Portfolio — Victor Morvan

Portfolio numérique réalisé dans le cadre du BUT Science des Données (parcours EMS) à l'IUT de Vannes.

Ce site présente mes Situations d'Apprentissage et d'Évaluation (SAÉ) réparties dans les quatre blocs de compétences du référentiel national (Traiter, Analyser, Valoriser, Modéliser), ainsi que mon expérience de stage et mon projet professionnel orienté Sport Data Analyst.

🔗 **Site en ligne :** https://victormorvan.github.io/portfolio-victor-morvan/
