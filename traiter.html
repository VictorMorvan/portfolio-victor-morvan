Conception et implémentation d'une base de données — Cas CompuDistri

Conception et implémentation d'une base de données relationnelle pour le cas métier CompuDistri, une entreprise de distribution informatique. Cycle complet : dictionnaire de données (code, libellé, nature, stabilité, domaine, commentaire), dépendances fonctionnelles, MCD sur Looping et IBEasy+, intégration MS Access via VBA et rédaction de 42 requêtes SQL.
